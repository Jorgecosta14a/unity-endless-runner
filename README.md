2. Requisitos Técnicos
Versão do Unity: Unity 6 (6000.0.45f1)

3. Descrição do Jogo
Desenvolvimento de um jogo estilo "Endless Runner" 3D.O jogador desloca-se continuamente num eixo,
evitando obstáculos dinâmicos,Funcionalidades implementadas incluem sistema de física com Rigidbody,
animações do personagem.

4. Jogabilidade
Objetivo: Sobreviver o máximo de tempo possível desviando dos obstáculos para acumular a maior pontuação.
Controlos: teclas (A,D) para mover a personagem lateralmente.
Regras: A colisão com um obstáculo causa o Game Over imediato. O jogador pode carregar no botão "Tentar de Novo"
 para reiniciar a partida.

5. Como Abrir o Projeto
   Abrir o Unity Hub
   Selecionar "Open" e escolher a pasta raiz deste repositório (`Endless-runner`).
   Dentro do Unity, navegar até à pasta `Assets/Scenes` e abrir a cena principal.
   Clicar no botão Play.

6. Assets Multimédia
Texturas e Modelos: Foram utilizados pacotes Low Poly para manter uma coerência visual e garantir boa performance.
As texturas estão em formato leve (.png/.jpg).
